Add Optio Dentistry patient education videos to your WordPress site.

== Description ==

This plugin allows you to easily add Optio Dentistry patient education videos to your WordPress site.

After activating the plugin, Optio Dentistry Video and Optio Dentistry Library blocks will be added to the WordPress editor. You will also be able to select text or other content and link to a video.

If your server configuration supports it, additional metadata will be included with embedded videos giving your site an SEO boost.

An [Optio Dentistry subscription](https://www.optiopublishing.com/order-dental-videos) is required.

== Installation ==

= Install from your WordPress dashboard =

1. Select "Plugins" from the left side menu and press "Add New"
2. Search for "Optio Dentistry"
3. Press "Install Now"
4. Activate Optio Dentistry from your plugins page
5. Edit the page or post where you would like to add videos
6. Add an Optio Dentistry Video or Optio Dentistry Library block

= Install from WordPress.org =

1. Download the Optio Dentistry plugin
2. Uncompress and upload the "optio-dentistry" folder into your "wp-content/plugins" folder using FTP
3. Activate Optio Dentistry from your plugins page
4. Edit the page or post where you would like to add videos
5. Add an Optio Dentistry Video or Optio Dentistry Library block
